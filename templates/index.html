SurfLog Journal

SurfLog is a shared surf journal for surf sessions in Denmark. It combines user notes with hourly weather and wave data from Open-Meteo so surfers can compare forecast conditions with what actually was experienced in the water.

Add New Session Open Journal

Objective

Build a simple and practical surf log that helps surfers learn from previous sessions and improve decision-making for future forecasts, making the chase a little more reliable.

How To Use

  1. Create a session in New Log.
  2. Write notes about what you observed in the water. See Surf Spots.
  3. Review the entry in Journal with hourly marine and weather data.

Current Scope

The current surf spots and workflow focus on the Danish coast. The app is intentionally simple and local-first while core features are being built.
